# Four-in-one-game
Годовой проект по информатике для конференции «Информационные технологии», которая проходила в ФМЛ №239 при поддержке JetBrains. Получил диплом второй степени. Реализованы четыре 3D игры на Unity. Сам код не публикую по личным причинам
